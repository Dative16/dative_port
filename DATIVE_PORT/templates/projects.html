AI-Powered Microscope for Malaria Detection

An innovative solution that combines hardware and software to automate malaria diagnosis. The system uses a custom-built microscope with a digital camera attachment and machine learning algorithms to detect and count malaria parasites in blood samples.

Technologies Used
  • Python
  • TensorFlow
  • OpenCV
  • Raspberry Pi
  • Flask
Key Features
  • Automated parasite detection and counting
  • Web-based interface for technicians
  • Patient record management
  • Statistical reporting
  • High accuracy (95%+) in detection
